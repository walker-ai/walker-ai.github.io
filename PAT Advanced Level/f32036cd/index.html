#1064.Complete Binary Search Tree

Problem Description

A Binary Search Tree (BST) is recursively defined as a binary tree which has the following properties:

  • The left subtree of a node contains only nodes with keys less than the node’s key.
  • The right subtree of a node contains only nodes with keys greater than or equal to the node’s key.
  • Both the left and right subtrees must also be binary search trees.

A Complete Binary Tree (CBT) is a tree that is completely filled, with the possible exception of the bottom level, which is filled from left to right.

Now given a sequence of distinct non-negative integer keys, a unique BST can be constructed if it is required that the tree must also be a CBT. You are supposed to output the level order traversal sequence of this BST.

Input Specification

Each input file contains one test case. For each case, the first line contains a positive integer $N$ ($\leq 1000$). Then $N$ distinct non-negative integer keys are given in the next line. All the numbers in a line are separated by a space and are no greater than 2000.

Output Specification

For each test case, print in one line the level order traversal sequence of the corresponding complete binary search tree. All the numbers in a line must be separated by a space, and there must be no extra space at the end of the line.

Preliminary knowledge

满二叉树

1、第 $i$ 层的节点数目为 $2^i$。

2、节点总数和深度的关系：$n = \sum\limits_{i=0}^k 2^i = 2^{k + 1} - 1$

完全二叉树

1、具有 $n$ 个节点的完全二叉树的深度（根节点的深度定义为 $0$）为 $k = \lfloor \log_2 n \rfloor$。

2、最后一层的节点数为：$n - (2^k - 1) = n + 1 - 2^k$

3、左子树的节点数为（总结点个数为 $n$）：

$$ l(n) = \begin{cases} n - 2^{k-1}, & n + 1 - 2^k\leq 2^{k-1} \\ 2^k - 1, & n + 1 - 2^k > 2^{k-1} \end{cases} $$

因为最后一层全部都在左子树，右子树为满二叉树，高度为 $k-1$。
