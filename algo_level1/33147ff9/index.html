前置知识

这里记录一下常用的位运算操作

n >> k & 1

表示 $n$ 的二进制表示中，第 $k$ 位数（从右往前数第几个数）是几，$k$ 从 $0$ 开始计数，例如：

二进制数 $(1010)_2$，$k = 0 \sim 3$

$k = 0$：1010 >> 0 & 1 = 0
$k = 1$：1010 >> 1 & 1 = 1
$k = 2$：1010 >> 2 & 1 = 0
$k = 3$：1010 >> 3 & 1 = 1

state | 1 << u

一般适用于给定 $n$ 个数，每一个数选与不选的情况，$u$ 一般从 $0$ 开始计数：$0 \sim n - 1$ 表示 $n$ 个数中的其中一个数。例如：

$n$ 为 $3$ 时，对于 $0, 1, 2$，若选 $1, 3$（即第 $0$, $2$ 个数），则过程为：

初始时状态state = 0，当前数为0（<< >> 运算符优先级高于位运算符& |）

随后可用 state >> i & 1 来判断第 $i$ 个数是否被选。

lowbit(x)

返回 $x$ 的最后一位 $1$，即实际上返回的是一个二进制数，这个二进制数的最高位 $1$ 是 $x$ 的最后一位 $1$ ，例如 $x = 1010$, lowbit(x) = (10)_2 ，lowbit(x)是树状数组的基本操作。

题目重述
