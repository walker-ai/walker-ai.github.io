能被整除的数

原题链接

题目描述

给定一个整数 $n$ 和 $m$ 个不同的质数 $p_1, p_2,\ldots,p_m$。

请你求出 $1\sim n$ 中能被 $p_1, p_2,\ldots, p_m$ 中的至少一个数整除的整数有多少个。

输入格式

第一行包含整数 $n$ 和 $m$。

第二行包含 $m$ 个质数。

输出格式

输出一个整数，表示满足条件的整数的个数。

数据范围

$1\leq m\leq 16,$
$1\leq n,p_i\leq 10^9$

前置知识

容斥原理

设 $S_1, S_2, \cdots, S_n$ 为有限集合，$\lvert S\rvert$ 表示集合 $S$ 的大小，则：

$$
\left \lvert \bigcup_{i=1}^n S_i \right \rvert = \sum_{i=1}^n \lvert S_i\rvert - \sum_{1\leq i < j\leq n}\lvert S_i\cap S_j \rvert + \sum_{1\leq i < j < k\leq n} \lvert S_i\cap S_j\cap S_k \rvert + \cdots + (-1)^{n+1}\lvert S_1\cap\cdots \cap S_n \rvert
$$

也可以简单的用文氏图表示：

以题目样例为例，$n = 10$，$p_1 = 2, p_2 = 3$，求 $1\sim 10$ 中满足能整除 $p_1, p_2$ 的数的个数，即 $2, 3, 4, 6, 8, 9$ 共7个。

记 $S_i$ 为 $1\sim n$ 中能整除 $p_i$ 的数构成的集合，根据容斥原理，所有的数的个数为各个集合的并集。

$S_1 = \lbrace 2, 4, 6, 8, 10\rbrace$，$S_2 = \lbrace 3, 6, 9\rbrace$，$S_1\cap S_2 = \lbrace 6\rbrace$，故 $S_1\cup S_2 = \lbrace 2, 3, 4, 6, 8, 9, 10\rbrace$

题目分析

  1. 每个集合实际上并不需要知道具体元素是什么，只需要知道这个集合的大小，大小为 $\lvert S_i\rvert = \lfloor \dfrac{n}{p_i}\rfloor$，比如本题中 $\lvert S_1\rvert = \lfloor\dfrac{10}{5}\rfloor = 2$，$\lvert S_2\rvert = \lfloor\dfrac{10}{3}\rfloor = 3$（质数倍数个数基本定理）
  2. 确定交集的大小。因为 $p_i$ 均为质数，这些质数的乘积就是他们的最小公倍数，$n$ 除以这个最小公倍数就是交集的大小，故 $\lvert S_1\cap S_2\rvert = \lfloor\dfrac{n}{p_1p_2}\rfloor = \lfloor\dfrac{10}{2\cdot 3}\rfloor = 1$
  3. 采用二进制表示每个集合的状态，每一位表示该位代表的状态选还是不选。以 $n = 4$ 为例，这里需要4个二进制位来表示每一个集合选与不选的状态，$\overbrace{1101}^{m=4}$，这里表示选中集合 $S_1, S_2, S_4$，故这个集合中元素个数为 $\lfloor \dfrac{n}{p_1p_2p_4}\rfloor$，因为集合个数是3个，根据公式，前面的系数为 $(-1)^{3-1} = 1$。所以到当前这个状态时，应是 $\text{res} = \text{res} + \lfloor\dfrac{n}{p_1p_2p_4}\rfloor$。这样，就可以将范围从 $0000$ 到 $1111$ 的每一个状态都表示出来了。
