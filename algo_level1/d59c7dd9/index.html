题目重述

给定一个按照升序排列的长度为 $n$ 的整数数组，以及 $q$ 个查询。对于每个查询，返回一个元素 $k$ 的起始位置和终止位置（位置从 0 开始计数）。如果数组中不存在该元素，则返回 -1 -1。

算法思想

  1. check(mid) 函数是判断 mid 是否满足红色区域的性质，若满足，则红色区域的边界点一定在 [mid, r] 上，故更新方式为 l = mid；若不满足，则说明红色边界点的位置在 [l, mid - 1]，故更新方式为 r = mid - 1。

  2. check(mid) 函数是判断 mid 是否满足绿色区域的性质，若满足，则绿色区域的边界点一定在 [l, mid] 上，故更新方式为 r = mid；若不满足，则说明绿色边界点的位置在 [mid + 1, r]，故更新方式为 l = mid + 1。

TIPS

  1. 二分的时候一定要想清楚答案在哪个范围内，最好手动模拟一下。

  2. 如果是第一种情况，由于C++默认向下取整，则需要做加1处理，即： mid = L + R + 1 >> 1。
