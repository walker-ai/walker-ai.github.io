递增三元组

发表于2022-02-25|更新于2022-02-25|算法学习蓝桥杯刷题笔记
|字数总计:1.1k|阅读时长:5分钟|阅读量:|评论数:

题目描述

给定三个整数数组

$A = [A_1, A_2,\ldots A_N],$
$B = [B_1, B_2,\ldots B_N],$
$C = [C_1, C_2,\ldots C_N],$

请你统计有多少个三元组 $(i,j,k)$ 满足：

$1\leq i,j,k\leq N$
$A_i < B_j < C_k$

输入格式

第一行包含一个整数 $N$。

第二行包含 $N$ 个整数 $A_1, A_2,\ldots A_N$。

第三行包含 $N$ 个整数 $B_1, B_2,\ldots B_N$。

第四行包含 $N$ 个整数 $C_1, C_2,\ldots C_N$。

输出格式

一个整数表示答案。

数据范围

$1\leq N\leq 10^5,$
$0\leq A_i, B_i, C_i\leq 10^5$

输入样例

输出样例

题目分析

由于给定的数据范围为 $10^5$，因此时间复杂度要控制在 $O(n\log n)$ 以内，也就是说最多只能枚举 $A, B, C$ 其中一个变量。如果枚举 $A$ 或者枚举 $C$，剩下两个变量的取值是有关系的，因此取法不能直接简单的根据乘法原理相乘；而如果枚举 $B$，$A$ 和 $C$ 的取法没有直接关系，所以可以直接用乘法原理相乘。

对于每个 $B_j$，需要求出在 $A$ 中有多少个数比 $B_j$ 小，同理对称的在 $C$ 中求出有多少个数大于 $B_j$。
