DAY20-贝茜的报复

发表于2022-01-26|更新于2022-01-26|算法学习2022春季每日一题
|字数总计:1.1k|阅读时长:4分钟|阅读量:|评论数:

原题链接

题目描述

农夫约翰和奶牛贝茜喜欢在业余时间互相出数学题。

约翰给贝茜出了一道相当难的问题，导致她没能解决。

现在，她希望通过给约翰出一道有挑战性的难题来报复他。

贝茜给了约翰一个表达式 $(B+E+S+S+I+E)(G+O+E+S)(M+O+O)$，其中包含七个变量 $B,E,S,I,G,O,M$（$O$ 是变量，不是零）。

对于每个变量，她给约翰一个列表，表中包含该变量可采用的最多 $20$ 个整数值。

她要求约翰计算，共有多少种给变量赋值的方法可以使得表达式的计算结果为偶数。

输入格式

第一行包含一个整数 $N$。

接下来 $N$ 行，每行包含一个变量和该变量的一个可能值。

每个变量至少出现 $1$ 次，最多出现 $20$ 次。

同一变量不会重复列出同一可能值。

输出格式

输出可以使得表达式的计算结果是偶数的给变量赋值的方法总数。

数据范围

$7\leq N\leq 140,$

所有变量的可能取值范围 $[−300,300]$
本题答案不会超出 int 范围。

输入样例

输出样例

样例解释

共有 $6$ 种可能的赋值方式：

注意，(2, 5, 7, 10, 1, 16, 19) 和 (3, 5, 7, 9, 1, 16, 19)，虽然计算结果相同，但是赋值方式不同，所以要分别计数。

题目分析/完整代码

解法一

二进制

一共有7个变量，每个变量最多有20个赋值，暴力枚举的话，一共有 $20^{7}$ 种情况，但由于题目要求求出计算结果为偶数的情况一共有多少种，因此可以不用枚举每一种情况。

这里涉及到一个公式，即

$$ (a + b)\ \%\ m = c \Rightarrow \begin{cases} a\ \%\ m = c \\ c\ \%\ m = c \\ \end{cases} $$

所以我们只需要讨论每个因子的取值情况就可以了。而三个因子中，重复出现的字母例如第一个因子中的 $S$ 和 $E$，第三个因子中的 $O$，出现偶数次的字母可以直接省略。

这样，就只需要考虑 $B$、$I$、$G$、$O$、$E$、$S$、$M$ 了。可以开一个哈希表，存储每个字母的奇、偶赋值情况，然后用二进制的方法枚举每一个字符选奇、偶的情况，对于每一种情况而言，根据乘法原理，将所有符合当前情况的方案数累乘起来，再累加至总答案即可。

解法二

DFS

思路和解法一一致，只是解法一是用二进制 + 枚举迭代的方式累加答案，而解法二是用DFS累加答案。
