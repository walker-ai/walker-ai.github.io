DAY19-COW

原题链接

题目描述

奶牛贝茜在她最喜欢的牧场中发现了一块石碑，上面刻有神秘的碑文。

碑文的文字似乎来自一种神秘的古代语言，可看作一个只包含 $C,O,W$ 三种字符的字符串。

尽管贝茜无法解密该文字，但是她很欣赏 $C,O,W$ 按顺序构成她最喜欢的单词 $COW$。

她想知道 $COW$ 在碑文中一共出现了多少次。

她不介意 $C,O,W$ 之间是否存在其他字符，只要这三个字符按正确的顺序出现即可。

她也不介意多个不同的 $COW$ 是否共享了一些字符。

例如，$COW$ 在 $CWOW$ 中只出现一次，在 $CCOW$ 中出现两次，在 $CCOOWW$ 中出现八次。

给定碑文中的文字，请帮助贝茜计算 $COW$ 出现的次数。

输入格式

第一行包含 $N$。

第二行包含一个长度为 $N$ 的字符串，其中只包含字符 $C,O,W$。

输出格式

输出给定字符串中 $COW$ 作为子序列（不一定连续）的出现次数。

数据范围

$1\leq N\leq 10^5$

题目分析/完整代码

设 $f_{i, j}$ 表示已经走完 $i$ 个字母（下标从1开始），并且当前的状态是 $j$ 的方案总数。一共有三个状态0，1，2，分别对应末尾字母为 $C, O, W$。

状态机为：

分析一下状态转移方程，以第 $i$ 个字母的取值情况来区分，

  1. 若第 $i$ 个字母为 $C$：

  • 如果不选第 $i$ 个字母，那么 $f_{i, 0} = f_{i - 1, 0}$，如果选第 $i$ 个字母，那么相当于多了一种方案，所以 $f_{i, 0} = f_{i - 1, 0} + 1$

  • 因为第 $i$ 个字母是 $C$，所以不管选不选方案总数不变，故 $f_{i, 1} = f_{i - 1, 1}$

  • 同上，因为第 $i$ 个字母是 $C$，不管选不选方案总数不变，故 $f_{i, 2} = f_{i - 1, 2}$

  1. 若第 $i$ 个字母为 $O$：

  • 同1，$f_{i, 0} = f_{i - 1, 0}$

  • 如果不选第 $i$ 个字母，那么 $f_{i, 1} = f_{i - 1, 1}$，如果选了，那么它除了要和前面的以 $O$ 为结尾的构成方案，同样也要和前面的以 $C$ 为结尾的构成方案，并且这种情况下方案数为 $f_{i - 1, 0}$，故 $f_{i, 1} = f_{i - 1, 1} + f_{i - 1, 0}$

  • 同1，$f_{i, 2} = f_{i - 1, 2}$

  1. 若第 $i$ 个字母为 $W$：

  • 同1，$f_{i, 0} = f_{i - 1, 0}$

  • 同1，$f_{i, 1} = f_{i - 1, 1}$

  • 同2，如果不选，那么 $f_{i, 2} = f_{i - 1, 2}$，如果选了，那么既要和前面以 $W$ 为结尾的构成方案，也要和前面以 $O$ 为结尾的构成方案，故 $f_{i, 2} = f_{i - 1, 2} + f_{i - 1, 1}$

这样，就可以将所有的状态方程解出来了。但有一点需要注意的是，最坏情况下，若三个字母的个数均为 $\dfrac{n}{3}$ 时，方案总数最大为 $\dfrac{10^{15}}{27}$，所以要开 long long。
